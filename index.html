Welkom bij Schaken

Deze website is volledig gemaakt in HTML, CSS en JavaScript. Ik heb heel wat tijd besteed aan een groot deel van de systemen.

Ik heb heel veel van de systemen zelf gemaakt, zoals onder anderen het drag-en-drop systeem. Er zijn ook wat van de functies geschreven door ChatGPT, dit is grotendeels de validatie van de zetten, omdat dat voor mijn hersencellen teveel informatie is om te verwerken in een script. Er zij in totaal 14 Javascript files, waarvan 6 geschreven door ChatGPT.

Een deel van deze scripts zijn gewoon puur wat helper functies, maar een groot deel is ook voor de functionaliteit van het spel.

Er zijn ook hele specifieke functionaliteiten die niet geimplementeerd zijn zoals onder anderen, rokeren, en passant slaan.

In dit project heb ik gebruik gemaakt van ES6 functies, zoals arrow functies, import en export, promises, de spread operator, om maar wat op te noemen.
